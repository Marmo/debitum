Avec Debitum, vous pouvez suivre toutes sortes de reconnaissances de dette, qu’il s’agisse d’argent ou d’objets prêtés. Ainsi, vous n’oublierez plus jamais si votre ami vous a déjà rendu ce livre ou vous ne vous disputerez plus sur la somme que vous devez à votre collègue pour le café.

Aucun service en ligne n’est impliqué, vos données sont donc enregistrées uniquement sur votre appareil (en ce qui concerne cette application).

Usage

Creating Transactions

Pour prendre note de l’argent ou des objets prêtés, créez une transaction à l’aide du bouton rouge. Une boîte de dialogue apparaît, dans laquelle vous pouvez saisir des infos tels que le montant, la description ou la date, et basculer entre une transaction d’argent et une transaction d’objet (utilisez le bouton à droite du champ de saisie du montant).

Transaction Lists

Choisissez l’affichage des transactions en bas de la page :

People

Transactions récapitulées par personne, classées par date de dernière transaction

  • Touchez une ligne pour filtrer l’argent et les objets par cette personne
  • Créez des personnes à partir du menu ici ou avec le bouton « + » en ajoutant une nouvelle transaction
Money

Affiche uniquement les transactions monétaires, par date (en ordre décroissant). Pour créer un remboursement, appuyez longuement sur la transaction initiale et sélectionnez l’élément de menu rembourser. Cela créera une transaction de remboursement correspondante. Pour savoir pourquoi il n’existe pas de fonctionnalité « marquer comme rendu » comme pour les objets, voir ici.

Items

Affiche uniquement les objets prêtés, par date (en ordre décroissant). Pour marquer les objets comme rendus, appuyez longuement sur eux et choisissez « Marquer comme rendu » dans la barre de menu. La liste peut être filtrée par objets rendus, non rendus ou tous les objets.

Appuyez longuement sur une ligne de n’importe quelle liste pour afficher son menu contextuel. Modifiez, supprimez ou sélectionnez plusieurs lignes de cette façon. Dans les listes « Personne » et « Argent », la somme de toutes les lignes sélectionnées est affichée.

Pour créer une copie d’une transaction existante ou en utiliser une comme préréglage, sélectionnez-la et appuyez sur le bouton « + ». La boîte de dialogue de création de transaction sera préremplie avec les valeurs de la transaction sélectionnée.

Linking Contacts

La boîte de dialogue « Modifier une personne » permet de lier des contacts téléphoniques existants à des personnes. Appuyez sur « Aucun contact lié » et choisissez-en un. Actuellement, le seul effet est que la photo du contact (si elle est présente) est affichée dans la liste « Personne ».

Backup and restore

Sauvegarde et restaure la base de données (de toutes les transactions et personnes) à partir des paramètres. Veillez à déplacer les fichiers de sauvegarde avant de désinstaller l’application (ou d’effacer les données de l’application), car la sauvegarde est enregistrée dans le dossier de données de l’application sur le stockage externe, qui est supprimé lors de la désinstallation ou l’effacement des données de l’application.

Permissions

Read contacts

Utilisé pour relier les contacts téléphoniques aux personnes. Vous pouvez refuser sans crainte cette autorisation et utiliser Debitum sans relier les contacts.
