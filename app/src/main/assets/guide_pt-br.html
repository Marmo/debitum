Com o Debitum você pode rastrear todos os tipos de débitos, seja em dinheiro ou itens emprestados. Desta forma, você nunca mais esquecerá se seu amigo já lhe devolveu aquele livro ou ficará na dúvida sobre o quanto você deve ao seu colega pelo café.

Não há nenhum serviço online envolvido e seus dados são salvos apenas no seu aparelho.

Modo de uso

Criar transação

Para anotar valor de dinheiro ou itens emprestados, você precisa criar uma transação usando o botão vermelho flutuante. Lá você pode digitar os detalhes como quantidade, descrição ou data e fazer alteração entre transação em dinheiro ou item (usando o botão à direita do campo de entrada de valor).

Lista de transações

Através da barra de navegação embaixo você pode escolher diferentes vistas de suas transações:

Pessoas

categorizado por pessoa, ordenadas por data da última transação

  • você pode tocar numa linha para ordenar a lista com dinheiro e itens por essa pessoa
  • você pode criar uma nova pessoa via menu nesta tela ou usando o botão + ao criar uma nova transação
Dinheiro

Shows only monetary transactions, ordered by date (descending). If someone repays a debt to you, you should create a new transaction for this. You can long-tap the initial transaction and select the repay menu item, which will create a matching repay transaction for you. For details why there is no "mark returned" functionality like for items see here.

Itens

Mostrar apenas itens emprestados, ordenados pela data (decrescente). Os itens podem ser marcados como devolvidos via clique longo e logo depois escolhendo a opção "marcar como devolvido". Depois a lista pode ser filtrada por itens devolvidos/não devolvidos/tudo.

You can get a context menu in every list by long-pressing a row. From there you can edit or delete items (also multiple rows can be selected this way). In the Person- and the Money-List the sum of all selected rows is displayed in the contextual action bar.

Linking Contacts

In the Edit-Person-Dialog you can link a contact from your phone to each person. To do this, click on the input labelled "no contact linked" and choose a contact. Currently, the only effect is, that the contact's photo (if present) will be shown in the person list.

Cópia de segurança e restauração

Na tela de configurações, você pode fazer um Backup e restaurar a base de dados (contém todas as transações e pessoas). Certifique-se de mover os arquivos do Backup para um lugar seguro antes de desinstalar o App ou limpar os dados do App, pois o Backup é salvo em pasta de dados no armazenamento externo, que é excluído ao desinstalar/limpar os dados do App.

Persmissions used

READ_CONTACTS (read contacts)

This is used for linking phone contacts to persons. You can safely deny this permission and use Debitum without linking contacts.
