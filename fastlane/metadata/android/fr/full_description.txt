Avec Debitum, vous pouvez suivre toutes sortes de reconnaissances de dette, qu'il s'agisse d'argent ou d'objets prêtés. Ainsi, vous n'oublierez plus jamais si votre ami vous a déjà rendu tel livre et vous ne vous disputez plus sur la somme que vous devez à votre collègue pour le café.

Le cas d'utilisation prévu est la gestion des (petites) dettes personnelles et des objets prêtés. Pour garder les choses simples, les choses comme les intérêts, les échéances, les frais, etc. ne feront pas partie intégrante de l'application (bien que la description puisse être utilisée à cette fin, mais je recommande généralement d'utiliser des applications plus spécialisées dans ce cas).

Vous pouvez sauvegarder et restaurer la base de données où sont stockées les personnes et les transactions.

Aucun service en ligne n'est impliqué, vos données sont donc sauvegardées uniquement sur votre appareil (en ce qui concerne cette application).

<i>Mots-clés : dette, prêt, emprunt, IOU, debt, lend</i>
